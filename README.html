Bit Vector

Bit Vector is a simple data Structure that is created in c++

How to get started:

Load the .sln file in visual studio to run the project.
The project contains only one class namd as BitVector. In the constructor you pass the number of bits that you want to allocate.
you can then set all bits to zero or one using set1 and set2 function. You can also set a particular bit by using set function and providing the bit number and value as parameter.
Sample code to use this class has been given in main function.
